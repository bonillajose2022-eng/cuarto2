# 🌟 English Fun Zone — 4th Grade Interactive English Learning

A fully interactive, visually rich, and kid-friendly English learning website for **4th grade students**. Built with pure HTML, CSS, and JavaScript — no frameworks required. Ready to deploy on **GitHub Pages**.

---

## 🚀 How to Deploy on GitHub Pages

1. **Create a new repository** on GitHub (e.g., `english-fun-zone`)
2. **Upload all project files** to the repository root:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch and `/ (root)` folder
5. Click **Save**
6. Your site will be live at: `https://yourusername.github.io/english-fun-zone/`

## 🎮 Interactive Activities Included

- **Multiple Choice** — Choose the correct answer from options
- **Fill in the Blank** — Type the missing word in sentences
- **True or False** — Decide if statements are correct
- **Sentence Ordering** — Drag and drop (or use arrows) to reorder steps
- **Matching Game** — Click to match sentence halves
- **Unscramble the Sentence** — Click words to build correct sentences
- **Memory Card Game** — Flip cards to find matching present/past verb pairs
- **Error Correction** — Find and fix incorrect connectives
- **Story Fill** — Complete a story using BE and HAVE
- **Cause & Effect Matching** — Match environmental causes to effects

## 🛠️ Technical Details

- **Pure HTML5, CSS3, JavaScript** — no frameworks or libraries
- **Google Fonts**: Fredoka One + Nunito
- **Responsive design** — works on desktop, tablet, and mobile
- **CSS Animations**: floating hero, confetti, card flip, fadeIn
- **Drag & Drop API** for sentence ordering
- **3D card flip** for memory game
- **YouTube embed** via modal for videos
- **Accessible**: semantic HTML, aria labels, keyboard navigation

---

## 👩‍🏫 For Teachers

- All exercises are **auto-corrected** with immediate feedback
- Students receive visual feedback: ✅ correct (green), ❌ wrong (red)
- Each wrong answer **reveals the correct answer** after checking
- Exercises can be **re-attempted** (refresh page or use Reset buttons)
- Score tracking is **session-based** (resets on page refresh)
